# Интеллектуальный агент для прогнозирования успеваемости студентов

##  Описание проекта
Разработка интеллектуального агента для анализа данных образовательного процесса, прогнозирования успеваемости студентов и классификации уровня академического риска. Система выполняет обработку данных о посещаемости, текущих оценках и результатах сессии для построения прогнозных моделей.

##  Функциональность
- **Многоклассовая классификация** (сдал/пересдача/комиссия/отчислен)
- **Регрессия** (прогноз среднего балла)
- **Авторизация** с разделением ролей (студент/деканат)
- **Персонализированные рекомендации** для студентов
- **Отчеты для деканата** со статистикой и списками группы риска

##  База данных
- **СУБД:** MySQL
- **Таблицы:** students, subjects, schedule, attendance, session
- **Данные:** 25 студентов, 10 дисциплин, расписание на 2 недели

##  Модели машинного обучения
- **Классификация:** Logistic Regression, Random Forest, XGBoost, CatBoost
- **Регрессия:** Linear Regression, Ridge, Random Forest, XGBoost
- **Лучшая модель классификации:** CatBoost (F1-macro: 0.83)
- **Важность признаков:** количество пересдач (50.5%), посещаемость лабораторных, процент сданных предметов
